About This Blog

This Personal Blog Website is built using Python Flask and enhanced with AI/ML features. It serves as a mini-project for AI/ML Engineering students, demonstrating how Natural Language Processing can be integrated into a real web application.

Users can create, edit, delete, and search blog posts through a clean, responsive interface. Behind the scenes, every post is automatically analysed by two AI modules.

AI / ML Features

1. Sentiment Analysis

Library: TextBlob

How it works: TextBlob uses a pre-trained sentiment lexicon (a dictionary of words with associated polarity scores) under the hood. When you submit a blog post, the library:

  1. Tokenises the text into sentences and words.
  2. Looks up each word's polarity in the lexicon.
  3. Averages the polarities to produce a single polarity score between -1.0 (very negative) and +1.0 (very positive).
  4. The score is then mapped to a label:
    • Positive — score > 0.1
    • Negative — score < -0.1
    • Neutral — score between -0.1 and 0.1
Example: The text "I love machine learning, it is amazing!" yields a polarity of approximately 0.62 → labelled as Positive.

2. Keyword Extraction (RAKE)

Library: rake-nltk (Rapid Automatic Keyword Extraction)

How it works: RAKE is an unsupervised, domain-independent algorithm that:

  1. Splits the text at stopwords (common words like "the", "is", "and") and punctuation to identify candidate keyword phrases.
  2. Builds a word co-occurrence matrix from those phrases.
  3. Scores each candidate phrase using the formula: score = word_degree / word_frequency.
  4. Ranks the phrases by score and returns the top-5.
Example: From the text "Deep learning models use convolutional neural networks for image classification tasks", RAKE might extract: convolutional neural networks image classification tasks deep learning models
